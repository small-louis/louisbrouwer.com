# louisbrouwer.com

Personal portfolio website. Static HTML and CSS, hosted on GitHub Pages.

## Structure

- `index.html` — Home page with project grid and about section
- `kinetic-canvas.html`, `supergreen.html`, `level.html`, `thesis.html`, `pluto.html`, `rivan.html` — Individual project pages
- `boog.html` + `boog-*.html` — Furniture design portfolio (9 sub-pages)
- `reading.html` — Reading list
- `images/` — Project images and media organised by project
- `*.pdf` — Thesis, CV, and project booklets with embedded PDF viewers

## Deployment

Pushes to `main` deploy automatically via GitHub Pages. Custom domain configured through `CNAME` file.
